# Employee Project & Expense Management Frontend

React + Vite + Tailwind CSS frontend for the FastAPI backend.

## Run

```powershell
cd frontend
npm.cmd install
npm.cmd run dev
```

Create `frontend/.env` from `.env.example` when the API URL differs:

```text
VITE_API_BASE_URL=http://localhost:8000/api/v1
```
